# OpenTenBase 小白安装指南

## 这是什么？

OpenTenBase 是一个分布式数据库（基于 PostgreSQL 增强）。你可以把它理解成一个"超级版 PostgreSQL"，能处理海量数据。

---

## 怎么安装？

**只需要复制这一条命令，粘贴到服务器终端，回车：**

```bash
curl -sSL https://repo.blackevil217.com/scripts/opentenbase.sh | sudo bash
```

就这样！脚本会自动帮你完成所有事情：
- ✅ 自动检测你的系统
- ✅ 自动下载安装软件
- ✅ 自动配置好一切
- ✅ 自动启动数据库

---

## 安装后怎么用？

安装完成后，数据库已经在运行了。你可以直接连接：

```bash
psql -h 127.0.0.1 -p 5432 -U opentenbase -d postgres
```

看到 `postgres=>` 提示符就说明成功了！输入 SQL 语句试试：

```sql
SELECT version();
```

---

## 常用操作

| 操作 | 命令 |
|------|------|
| 连接数据库 | `psql -h 127.0.0.1 -p 5432 -U opentenbase -d postgres` |
| 查看集群状态 | `opentenbase_ctl status` |
| 停止集群 | `opentenbase_ctl stop` |
| 启动集群 | `opentenbase_ctl start` |

## 常见问题

### Q: 安装失败怎么办？

**A:** 如果 CDN 链接慢或失败，换用 GitHub 直连：
```bash
curl -sSL https://raw.githubusercontent.com/CDUESTC-OpenAtom-Open-Source-Club/OpenTenBase-Packages/main/scripts/opentenbase.sh | sudo bash
```

### Q: 支持哪些系统？

**A:**
- Ubuntu 20.04 / 22.04 / 24.04
- Debian 11 / 12 / 13
- Rocky Linux 8 / 9
- AlmaLinux 8 / 9
- CentOS Stream 8 / 9
- Fedora 40
- openEuler 22.03
- 华为云 EulerOS 2.0

### Q: 需要联网吗？

**A:** 是的，安装过程需要联网下载软件包。安装完成后可以离线使用。

### Q: 需要多少内存？

**A:** 最低 2GB，推荐 4GB 以上。

### Q: 数据存在哪里？

**A:** 数据文件在 `/var/lib/opentenbase/`，配置文件在 `/etc/opentenbase/`。

### Q: 怎么卸载？

**A:**
```bash
curl -sSL https://repo.blackevil217.com/scripts/uninstall.sh | sudo bash
```
